Cách cài đặt Zend Framework 2

Cài đặt Zend Framework 2 Sekeleton Application

Đầu tiên bạn nên tải source của Zend Framework 2 Sekeleton Application. Đây là một ứng dụng mẫu được setup theo cấu trúc của Zend Framework 2 (Điều này là không cần thiết nến bạn đã nắm rõ cấu trúc thư mục của ZF2, Tất nhiên nếu sử dụng ZF2 Application Sekeleton thì sẽ nhanh hơn).

  1. Tải ZendSekeletonApplication bằng git. Nếu máy bạn đã có git thì bạn có thể dùng lệnh: git clone https://github.com/zendframework/ZendSkeletonApplication.git [thu_muc_cai_dat] Lệnh này sẽ thực hiện sao chép một repository (trong repo đó sẽ chứa toàn bộ source code) từ github về máy của bạn. Nếu bạn không nhập thu_muc_cai_dat thì git sẽ tự động tạo một thư mục mới với tên là tên của repository trong trường hợp này sẽ là: ZendSkeletonApplication vào thư mục mà dấu nhắc lệnh đang đứng.

Có rất nhiều cách để bạn có thể cài đặt được Zend Framework 2. Mình sẽ hướng dẫn một số cách thường được sử dụng.
